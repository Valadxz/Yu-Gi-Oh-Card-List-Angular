## :toolbox: Getting Started

### :running: Run Locally

Clone the project

```bash
https://github.com/Valadxz/Yu-Gi-Oh-Card-List-Angular/
```
Primero
```bash
https://github.com/Valadxz/Yu-Gi-Oh-Card-List-Angular.git
```
Instalar dependencias
```bash
npm install
```
Iniciar el servidor
```bash
ng serve -o
```
